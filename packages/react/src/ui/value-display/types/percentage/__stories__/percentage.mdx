### percentage

Renders a percentage with a chart

#### Value type

```
type value = number | undefined
// or
type value = {
    percentage: number | undefined
    label?: string
    placeholder?: string
}
```

#### PercentageInputAsObject

<Canvas
  of={PercentageStories.PercentageInputAsObject}
  source={{
    code: `
render: (item) => ({
    type: 'percentage',
    value: 42
})
`,
  }}
/>

#### PercentageValueInputAsObject

<Canvas
  of={PercentageStories.PercentageValueInputAsObject}
  source={{
    code: `
render: (item) => ({
    type: 'percentage'
    value: {
      percentage: 42
    }
})
`,
  }}
/>

#### With label

<Canvas
