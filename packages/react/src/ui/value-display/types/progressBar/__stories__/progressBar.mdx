### progressBar

Renders a progress bar visualization for displaying completion or progress
values.

#### Value type

```
type value = number | undefined
// or
type value = {
    value: number | undefined
    max?: number
    label?: string
    color?: string
    placeholder?: string
}
```

#### ProgressBarSimple

<Canvas
  of={ProgressBarStories.ProgressBarSimple}
  source={{
    code: `
render: (item) => ({
    type: 'progressBar',
    value: 75
})
`,
  }}
/>
