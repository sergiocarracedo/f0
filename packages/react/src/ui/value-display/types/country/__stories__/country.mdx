### country

Renders a country flag avatar with the country name. The country name is
automatically translated based on the i18n context, or can be provided via the
label property.

#### Value type

```
type value = {
    code: CountryCode
    label?: string
}
```

#### CountryType

<Canvas
  of={CountryStories.CountryType}
  source={{
    code: `
render: () => ({
    type: 'country',
    value: {
        code: 'es'
    }
})
`,
  }}
/>

#### CountryTypeWithLabel

<Canvas
  of={CountryStories.CountryTypeWithLabel}

    code: `
